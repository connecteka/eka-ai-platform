# Webhooks & Integrations

### PayU webhooks

PayU webhooks are the source of truth for activation. Assume at-least-once delivery. Make processing idempotent.

### WhatsApp notifications

Use Twilio for WhatsApp delivery. Track delivery failures.

### Email

Use Resend for invoice and system emails. Track bounces and spam complaints.
