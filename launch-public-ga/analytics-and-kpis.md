# Analytics & KPIs

If you can’t measure it, you can’t fix it. Keep GA metrics simple.

### Activation funnel

Track these events:

* Visit landing page
* Signup started
* Signup completed
* Email verified
* Workshop created
* First job card created
* First invoice generated

Define your activation goal:

* **Activated** = workshop created + first job card.

### Product usage

* Job cards created per workshop per week
* Invoices generated per workshop per week
* AI chat sessions per workshop per week

### Reliability

* API error rate
* P95 latency
* Crash-free sessions
* Notification delivery failures

### Support signals

* Tickets per 100 signups
* Top 5 issue categories
* Time to first response

### Week-1 targets (fill these)

* Activation rate: \_\_%
* D1 retention: \_\_%
* D7 retention: \_\_%
* Ticket rate: \_\_ per 100 signups
